== Description ==

**Duitku** is a payment gateway plugin for PipraPay that allows users to accept payments via Duitku (an Indonesian payment gateway). Supports various payment methods including credit cards, e-wallets, and bank transfers.

**Key Features:**
* Multiple payment methods support
* Real-time transaction verification
* Merchant dashboard integration
* Secure payment processing
* Invoice and receipt management

**Why Use Duitku?**
Duitku provides secure and reliable payment processing for Indonesian businesses with multiple payment method options.
