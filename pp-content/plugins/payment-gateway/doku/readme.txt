== Description ==

**DOKU** is a payment gateway plugin for PipraPay that enables users to accept payments via DOKU (PT Fintek Karya Nusantara). DOKU provides comprehensive payment solutions for Indonesian merchants with support for multiple payment channels.

**Key Features:**
* Multi-channel payment processing
* Virtual account support
* E-wallet integration
* Real-time transaction status
* Comprehensive merchant dashboard
* Robust security features

**Why Use DOKU?**
DOKU is a trusted fintech provider in Indonesia offering secure, scalable payment solutions with excellent support for merchants.
