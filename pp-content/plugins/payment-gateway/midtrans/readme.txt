== Description ==

**Midtrans** is a payment gateway plugin for PipraPay that allows users to accept payments via Midtrans (a leading Indonesian payment gateway). Supports multiple payment methods including credit cards, e-wallets, bank transfers, and more.

**Key Features:**
* Support for multiple payment methods
* Sandbox and live mode
* Transaction verification
* Invoice management
* Customer data handling

**Why Use Midtrans?**
Midtrans provides secure payment processing for Indonesian merchants with support for all popular payment methods.
